NAME

Doctool::Method - Data container for GDScript methods

METHODS

new( )

Constructs a Doctool::Method object and returns a reference to it.

add_arg( ARG )

Adds ARG to the list of method arguments. ARG must be a Doctool::Arg object.

as_text( )

Returns a string describing the method. An example would be int max(int a, int b). The string also factors in default values, so the following could come out: bool open(String file, int mode = 0).

If the const attribute is set, "const" is appended to the end of the string output. For example, with the const attribute the output could look like this: int size() const.

If the virtual attribute is set, "virtual" is prepended to the beginning of the string output. For example, with the virtual attribute the output could look like this: virtual int size().

clear_args( )

Clears the list of arguments, making array empty.

find_arg( FIELD, PATTERN )

Attempts to find the argument with the given FIELD that matches PATTERN.

If the pattern matches, then the Doctool::Arg object that matches is returned. Only the first match is returned.

get( ATTR )

Obtains the value associated with ATTR. ATTR must be one of the attributes defined in "ATTRIBUTES".

set( PARAM, VALUE )

Sets the value associated with ATTR. ATTR must be one of the attributes defined in "ATTRIBUTES".

remove_arg( INDEX )

Removes the argument specified by INDEX from the list.

Internally, this method uses splice() to remove the specified index from an array.

set_arg( INDEX, VALUE )

Sets the method argument at INDEX to VALUE. VALUE must be an object of Doctool::Arg.

ATTRIBUTES

args

A list of the function arguments. Each element must be an object of Doctool::Arg. Do not attempt to set this attribute directly through set(). Instead, use set_arg() and add_arg() to manipulate the argument list.

brief

A brief one-line description of the method.

const

A boolean to describe whether the method is const (doesn't modify class variables).

description

A full description of the method.

name

The name of the method.

returntype

The return type of the method.

returnvalue

A description of the return value.

virtual

A boolean to describe whether the method is virtual.
