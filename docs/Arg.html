NAME

Doctool::Arg - Data container for generic typed arguments

METHODS

new( )

Constructs a Doctool::Arg object and returns a reference to it.

as_text( )

Returns a string which describes the argument. Without a default value the string will look something like this: int example. But with a default value the string can look like this: int example = 0.

get( ATTR )

Obtains the value associated with ATTR. ATTR must be one of the attributes defined in "ATTRIBUTES".

set( PARAM, VALUE )

Sets the value associated with ATTR. ATTR must be one of the attributes defined in "ATTRIBUTES".

ATTRIBUTES

These are the attributes which are recognized by set() and get().

default

The default value.

name

The name of the argument.

type

Argument type.
