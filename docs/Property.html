NAME

Doctool::Property - Data container for GDScript exported properties

METHODS

new( )

Constructs a Doctool::Property object and returns a reference to it.

get( ATTR )

Obtains the value associated with ATTR. ATTR must be one of the attributes defined in "ATTRIBUTES".

set( ATTR, VALUE )

Sets the value associated with ATTR. ATTR must be one of the attributes defined in "ATTRIBUTES".

ATTRIBUTES

These are the attributes which are recognized by set() and get().

brief

A brief one-line description of the property.

default

The property's default value.

description

Full description of the property.

getter

The property's getter function.

name

The name of the property.

setter

The property's setter function.

type

The property's type.
