NAME

Doctool::Signal - Data container for GDScript signals

METHODS

new( )

Constructs a Doctool::Signal object and returns a reference to it.

add_arg( ARG )

Adds ARG to the list of method arguments. ARG must be a Doctool::Arg object.

as_text( )

Returns a string representation of the signal. Supposing we have a signal named "test" with two arguments, one int named 'a' and one bool named 'b', the output would be test(int a, bool b).

get( ATTR )

Obtains the value associated with ATTR. ATTR must be one of the attributes defined in "ATTRIBUTES".

set( PARAM, VALUE )

Sets the value associated with ATTR. ATTR must be one of the attributes defined in "ATTRIBUTES".

set_arg( INDEX, ARG )

Replaces the argument at INDEX with ARG. ARG must be a Doctool::Arg object.

ATTRIBUTES

These are the attributes which are recognized by set() and get().

args

A list of Doctool::Arg objects which specify the parameters for the signal. Do not set this directly through set(). Instead use add_arg() or set_arg().

brief

A brief one-line description of the signal.

description

A full description of the signal.

name

The name of the signal.
